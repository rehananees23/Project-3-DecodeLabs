## 🎨 Visual System & Identity

This interface adheres to a strict visual rhythm to maintain a curated, organic, and trustworthy aesthetic:

### 1. Typography & Hierarchy
*   **Headlines (Editorial Authority):** `Playfair Display` (Serif) — Establishes a premium, heritage farm aesthetic, conveying authenticity and high-end curation.
*   **Body & Controls (Utility & Readability):** `Montserrat` (Sans-Serif) — Ensures clean, rapid readability for weights, prices, timelines, and interface labels.

### 2. Color Palette & Psychology
*   🟩 **Leaf Green (Primary):** Represents vitality, freshness, and life. Used on high-priority call-to-actions ("Enter the Harvest", "Add to Basket").
*   🟫 **Earthy Charcoal & Sand (Base):** Soft, grounding warm-gray and deep charcoal tones that avoid the stark, synthetic feel of pure white and pitch-black backgrounds.
*   🟨 **Harvest Gold (Accents):** Used strictly for premium features (like the "Best Value" subscription badge).
*   🚫 **No Red:** Purely positive visual reinforcement; avoids stressful "fast-food panic" red colors.

## ⚙️ Key UI Features ("The Trust Engine")

Rather than using basic templates, this interface features highly specialized components to tackle specific user hesitations:

### 🌦️ 1. Context-Aware Adaptive Banners
*   **The Component:** The **"Rainy Day Comforts"** card on the Product List page.
*   **The Logic:** Instead of static generic sales banners, the interface adapts to external user context (like weather or season), matching the customer's real-time mood with relevant, warming produce recommendations.

### 📍 2. Radical Transparency Systems
*   **The Component:** The **"Freshness Timeline"** and explicit **Harvest Details** (e.g., *"Harvested at Stone Barns Farm at 8:00 AM today"*).
*   **The Logic:** Combats user skepticism about grocery freshness. Giving the exact farm location and harvest time turns quality from a claim into proof.

### 🔄 3. Frictionless Micro-Subscriptions
*   **The Component:** Integrated **"Weekly Sub"** toggles directly on the product grid and detail cards.
*   **The Logic:** Encourages recurring orders without forcing the user to commit through complex sub-menus. The UI makes turning a one-time purchase into a weekly delivery as simple as a single toggle switch.

### 🔌 4. Eco-Social Value Badges
*   **The Component:** **"Carbon Neutral Delivery"** tracker showing exact CO2e savings.
*   **The Logic:** Appeals to the values of modern, eco-conscious organic consumers, making them feel positive about the delivery footprint.
